# film Front-end

This folder contains the main frontend code for the project. Below are the main code files you created (excluding the assets folder):

- **Controller**: Handles page logic and user interactions
  - age-gate.js
  - auth.js
  - home.js
  - particles.js
  - profile.js
  - ticket-confirmation.js
  - tickets.js
- **Model**:
  - ddata.js
- **css**: Stylesheets for the application
  - age-gate.css
  - auth.css
  - style-additions.css
  - style.css
  - ticket-confirmation.css
  - tickets.css
- **view**: HTML files for each page
  - auth.html
  - home.html
  - index.html
  - profile.html
  - ticket-confirmation.html
  - tickets.html
