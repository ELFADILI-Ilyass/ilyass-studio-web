# film Back-end

This folder contains the main backend code for the project. Below are the main modules you created (excluding the default app module):

- **auth**: Authentication logic and endpoints
- **locations**: Location management
- **payments**: Payment processing
- **tickets**: Ticket management
- **users**: User management

Each module contains its controller, service, and entity files. Example structure:

```
backend/src/
  auth/
    auth.controller.ts
    auth.module.ts
    auth.service.ts
  locations/
    locations.controller.ts
    locations.module.ts
    locations.service.ts
    entities/location.entity.ts
  payments/
    payments.controller.ts
    payments.module.ts
    payments.service.ts
    entities/payment.entity.ts
  tickets/
    tickets.controller.ts
    tickets.module.ts
    tickets.service.ts
    entities/ticket.entity.ts
  users/
    users.controller.ts
    users.module.ts
    users.service.ts
    entities/user.entity.ts
```
